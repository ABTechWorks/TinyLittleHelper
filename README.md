## Overview

**TinyLittleHelper** is a minimalistic device monitoring system. Track devices in real-time, see online/offline status, and register devices manually or automatically using the TinyLittleHelper client.

---

## Features

- User signup & login with email verification  
- Add devices manually (IP and/or MAC)  
- Automatic device registration via TinyLittleHelper client  
- Real-time device monitoring dashboard  
- Heartbeat updates every 30 seconds  
- Responsive, clean web interface
